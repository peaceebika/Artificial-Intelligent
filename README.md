# Machine Learning & Artificial Intelligence Projects

## Overview

This repository contains a curated collection of **Machine Learning (ML), Deep Learning (DL), and Reinforcement Learning (RL) projects**, developed primarily for **learning, experimentation, and teaching purposes**.

All projects were created using **Google Colab** and demonstrate both **classical AI techniques** and **modern learning algorithms**, ranging from decision trees and neural networks to reinforcement learning and game-playing agents.

---

## Repository Structure

├── Decision Tree # Decision tree models
├── MLP on Digits Dataset # MLP on sklearn digits
├── MLP_Optimizer_Comparison_MNIST # Optimizer comparison on MNIST
├── Maze Solver using DFS and BFS # Graph search algorithms
├── Naive Bayes Sentiment Classifier # NLP sentiment analysis
├── Three_State_MDP_Q_SARSA_REINFORCE # RL algorithms on simple MDP
├── Tic-Tac-Toe AI Using Minimax # Game-playing AI
├── classical_ml # Classical ML algorithms
├── deep_learning # Neural network models
└── reinforcement_learning # RL experiments

## Project Descriptions

### Decision Tree
Implementation of **Decision Tree classifiers**, focusing on:
- Feature splitting
- Tree depth control
- Overfitting vs. generalization

**Category:** Classical Machine Learning

---

### MLP on Digits Dataset
A **Multi-Layer Perceptron (MLP)** applied to a digits dataset.
- Feedforward neural networks
- Backpropagation
- Classification accuracy evaluation

**Category:** Deep Learning

---

### MLP Optimizer Comparison (MNIST)
Comparison of different **optimization algorithms** for training an MLP on the **MNIST dataset**:
- SGD
- Adam
- RMSprop

**Category:** Deep Learning

---

### Maze Solver using DFS and BFS
Maze-solving using:
- Depth-First Search (DFS)
- Breadth-First Search (BFS)

**Category:** Classical AI / Search Algorithms

---

### Naive Bayes Sentiment Classifier
A **Naive Bayes** classifier for sentiment analysis:
- Text preprocessing
- Probabilistic modeling

**Category:** NLP / Classical ML

---

### Three-State MDP (Q-Learning, SARSA, REINFORCE)
Reinforcement learning on a simple MDP using:
- Q-Learning
- SARSA
- REINFORCE

**Category:** Reinforcement Learning

---

### Tic-Tac-Toe AI using Minimax
Game-playing AI using the **Minimax algorithm**.

**Category:** Classical AI / Game Theory
